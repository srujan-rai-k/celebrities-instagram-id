MCQ ON IOT UNIT-2 WEEK-5 by NPTEL

1.Which of the following user interoerability problems is defined by the way how existing device services are cataloged to be discovered by device users?

a) Device sense disambiguation
b) Device categorization
c) Syntactic device interoperability
d) Semantic device interoperability

Answer: c

2.Which of the following defines "the meaning of a user's instruction sent from the user to a device is interpretable by the device exactly as the user originally means"?

a) Device sense disambiguation
b) Device categorization
c) Syntactic device interoperability
d) Semantic device interoperability

Answer: d

3.Which of the following is not used to generate a unique address in case of identification and categorization for device discovery?

a) Electronic Product Code
b) IPv4 addressing scheme
c) Universal Product Code
d) IPv6 addressing scheme

Answer: b

4.Service-oriented Computing(SOC)-based architecture is a popular approach for ______?

a) Device sense disambiguation
b) Device categorization
c) Syntactic device interoperability
d) Semantic device interoperability

Answer: c

5.According to the Collaborative Concept Exchange, which of the following solves the problem of cross-context syntactic interoperability problem?

a) Pearl syntax
b) XML syntax
c) JavaScript syntax
d) Lua syntax

Answer: b

6.Which of the following approach is used to solve the semantic interoperability problem?

a) Set of ontologies
b) Standards of Simple Object Access Protocol
c) Middleware technologies
d) XML Product Map

Answer: a

7.What does the collaborative conceptualization theory do?

a) Overcomes the limitation of ontology
b) Enables both the cross-context semantic and syntactic interoperability
c) Overcomes the limitation of ontology and enables the cross-context semantic interoperability
d) Overcomes the limitation of ontology and enables the cross-context syntactic interoperability

Answer: a

8.In the Universal Middleware Bridge architecture, which one of the following translates the local middleware's message into global metadata's message?

a) UMB Core
b) UMB Adapter
c) UMB Hander
d) UMB Manager

Answer: b

9.What is a program written Arduino IDE called?

a) Source Code
b) IDE code
c) Main
d) Sketch

Answer: d

10.In Arduino programming delay(100) results in a delay of _____

a) 100 seconds
b) 100 milliseconds
c) 100 microseconds
d) None of these

Answer: b

11.In Arduino programming, what does isSpace() do?

a) Analyze if a void is a white-space character
b) Analyze if a string is a white-space character
c) Analyze if a int is a white-space character
d) Analyze if a char is a white-space character

13.In Arduino programming, the function length() returns the length of the string without considering null?
