MCQ ON IOT UNIT-2 WEEK-3 by NPTEL

1.Which of the following is the standard for the Wireless-HART protocol?

a) IEEE 802.16
b) IEEE 802.15.4
c) IEEE 802.15.6
d) IEEE 802.15.2

Answer: b

2.Which of the following lacks a network layer?

a) HART
b) Wireless HART
c) Wired HART
d) None of these

Answer: c

3.By employing 15 channels of the ISM band, physical layer of HART achieves which one of the following?

a) Reliability
b) Reachability
c) Security
d) Scalability

Answer: a

4.Which of the following layers does not differentiate between Wireless HART and Wired HART?

a) Application Layer
b) Network Layer
c) Data Link Layer
d) Physical Layer

Answer: a

5.At the MAC layer which protocol applies CSMA channel access mode?

a) ZigBee
b) Bluetooth
c) HART
d) Wireless HART

Answer: a

6.Which of the following principles does Near-field communication(NFC) protocol operate?

a) Electromagnetic field
b) Magnetic Induction
c) Capacitive coupling
d) Electric field

Answer: b

7.Which of the following technology is based on Ad-hoc piconets?

a) RFID
b) NFC
c) ZigBee
d) Bluetooth

Answer: d

8.Which protocol is used for replacing traditionally used serial cables ?

a) RFCOMM
b) SDP
c) L2CAP
d) None of these

Answer: a

9.In India Z-wave operates in which frequency range?

a) 2.4 GHz
b) 865.2 MHz
c) 919.8 MHz
d) 922-926 MHz

Answer: b

10.Which of the following is correct for Z-wave and Zigbee?

a) Both uses 2.4GHz frequency range
b) Both uses DSSS modulation
c) Both technologies are mesh networks
d) Both uses Gaussian FSK

Answer: d

11.The physical layer in ISA 100.11A is derived from _______?

a) IEEE 802.15.4
b) IEEE 802.11g
c) IEEE 802.11a
d) IEEE 802.15.6

Answer: a

12.Which security mechanism is followed by ISA 100.11A?

a) 3DES
b) DES-56 encryption
c) AES-256 encryption
d) AES-128 encryption

Answer: d

13.Why nodes use a "watchdog" mechanism in a wireless ad hoc and sensor network scenario?

a) To detect misbehaving nodes
b) To ensure if other nodes do relay data packets
c) To ensure avoidance of malicious nodes
d) All of these

Answer: d

14.The security attack "eavesdropping" is a threat to which of the following layers?

a) Physical layer
b) Data link layer
c) Transport layer
d) Application layer

Answer: a

15."Denial of service" is mainly exhibited by which of the following nodes?

a) Failed nodes
b) Badly failed nodes
c) Malicious nodes
d) Selfish nodes

Answer: c
