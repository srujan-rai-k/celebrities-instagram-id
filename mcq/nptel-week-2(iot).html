MCQ ON IOT UNIT-2 WEEK-2 by NPTEL

1.Which of the following operations is true for MQTT?

a) Publish/subscribe
b) request/response
c) client/server
d) None of these

Answer: a

2.In MQTT to the CONNECT message from the client to initiate a connection, the Broker replies with which of the following?

a) ACK
b) GET
c) CONNECTED
d) CONNACK

Answer: d

3.Which of the following is true for the MQTT topic?

a) UTF-8 string
b) Case sensitive
c) Both of them
d) None of them

Answer: c

4.Which of the following is the protocol data transfer unit for CoAP?

a) Datagram
b) Packet
c) Frame
d) Bit

Answer: a

5.Which one of the following is not resource-constrained protocol?

a) MQTT
b) Co-AP
c) REST
d) DDS

Answer: c

6.Which of the following is not a feature of the CoAP protocol?

a) Web Transfer Protocol
b) M2M applications
c) Request-response protocol
d) Synchronous communication

Answer: d

7.Which of the following is not a feature of XMPP?

a) Central broker
b) Open standard
c) Encryption
d) Interoperability

Answer: a

8.In which of the following protocol message is sent over both TCP and UDP?

a) AMQP
b) MQTT
c) CoAP
d) XMPP

Answer: c

9.Which AMQP component receives message and routes them to queues?

a) Exchange
b) Queue
c) Bindings
d) Transfer

Answer: a

10.Which feature is responsible for interference-free access to the same channel at different times by multiple users or nodes in IEEE 802.15.4?

a) DSSS modulation
b) CSMA-CA multiplexing
c) Transmission of short packages
d) Greater channel bandwidth

Answer: b

11.Which of the following is functionality for RFD(Reduced Function Device) in IEEE 802.15.4?

a) Switching
b) Routing
c) Relaying
d) Sensing

Answer: d

12.Which of the following is not a functionality of Zigbee Device Objects(ZDOs)?

a) Device management
b) Managing requests to join a network
c) Interfacing and control services
d) Security

Answer: c

13.Which technique allows transceiving of IPv6 packets over IEEE 802.15.4 in 6LoWPAN?

a) Header compression
b) Address translation
c) Both of them
d) None of them

Answer: c

14.Which of the following is not a type of header in 6LoWPAN?

a) Dispatch
b) Mesh Addressing
c) Fragmentation
d) Flow label

Answer: d

15.Route Requests,Route Replies and Route Error are the operations for which routing protocol?

a) RPL
b) E-RPL
c) LOADng
d) None of them
